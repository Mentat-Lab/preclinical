## Category 2 Collection Command

Use this for the full Category 2 run:

```bash
./scripts/run-benchmark.sh --targets category2 --wait --concurrency 3
```

The script starts ChatGPT, Claude AI, and Gemini web in parallel, but forces each browser target to run one scenario at a time. This matches the BrowserUse Cloud account limit of 3 active sessions and avoids multiple concurrent scenarios fighting inside the same chatbot/profile target.

## 403 Handling

Any 401/403 from model calls is classified as `PROVIDER_AUTH`, non-retryable, and should fail the scenario/test run path rather than being treated as a successful transcript. Intent-analysis and response-validation helper LLMs now rethrow provider auth failures instead of silently falling back.
